How to install

The UMC IDE is available in 2 flavours: eclipse plugin and standalone application.

  • * Eclipse plugin: How to install information can be found in the readme.
  • * Standalone application:
    - Download for Windows x86 x64
    - Download for Linux x86 x64
    - Download for Mac OSX: x86 x64
    The application requires the Java runtime environment, so you need to install SUN's Java first (not openJDK or others) - Download Java.
