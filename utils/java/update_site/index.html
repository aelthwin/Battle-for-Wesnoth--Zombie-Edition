The Battle for Wesnoth UMC Development IDE

The plugin is a UMC IDE. Basically it offers some features that greatly help anybody who makes User Made Content for Wesnoth.

  • * It has wizards for new campaigns, scenarios, factions, eras, and everything else you might think of. You just complete the needed values and it generates the wml (with nice indentation).
  • * It has front-ends for launching wmltools (like wmllint, wmlscope, wmlindent on files/directories), the wesnoth game, and map editor.
  • * It has a specialized WML editor that has syntax highlighting, tag folding and outline, autocompletion, a macro "explorer", and so on.
  • * It has an 'upload addon' feature that uploads the specified project to the addons server.
  • For a full features list please check the dedicated page.
